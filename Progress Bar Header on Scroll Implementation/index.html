Progress Bar Header on Scroll

This page demonstrates a header with a progress bar that fills as you scroll down. The progress bar gives you visual feedback about your reading progress through the content.

Scroll down to see the progress bar in action!

How It Works

The progress bar is implemented using JavaScript that calculates the scroll percentage of the page. As you scroll down, the width of the progress bar increases proportionally to how far you've scrolled.

The formula used is:

scrollPercentage = (window.scrollY / (document.documentElement.scrollHeight - window.innerHeight)) * 100

Benefits of Progress Indicators

Progress indicators in web design serve several important purposes:

  • Provide visual feedback to users about their position in content
  • Help users estimate how much content remains
  • Improve user experience by reducing uncertainty
  • Can increase engagement with long-form content

Implementation Details

This implementation uses:

  • Fixed positioning for the header to keep it visible while scrolling
  • A simple div element for the progress bar with dynamic width
  • JavaScript event listener for the scroll event
  • CSS transitions for smooth animation
  • Responsive design principles for mobile compatibility

Customization Options

The progress bar can be easily customized:

  • Change the color by modifying the background-color property
  • Adjust the height by changing the progress container height
  • Modify the animation speed by adjusting the transition duration
  • Add different styles like gradients or rounded corners
