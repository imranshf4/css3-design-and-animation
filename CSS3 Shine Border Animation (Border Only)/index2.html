Shine Border Effect

CSS Code for Shine Border

.shine-border {
    position: relative;
    width: 300px;
    height: 200px;
    border-radius: 12px;
    padding: 2px;
    background: linear-gradient(115deg, 
        transparent 20%, 
        #A07CFE 30%, 
        #FE8FB5 50%, 
        #FFBE7B 70%, 
        transparent 80%
    );
    background-size: 200% 100%;
    background-position: -150% 0;
    animation: shine 3s infinite;
}

@keyframes shine {
    0% {
        background-position: -150% 0;
    }
    50% {
        background-position: 150% 0;
    }
    100% {
        background-position: -150% 0;
    }
}
This implementation uses pure CSS3 with linear-gradient and animation properties
